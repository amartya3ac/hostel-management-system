---

## ✨ Features

- 👤 **Student Registration**
- 📋 **Room Allocation**
- 🛏️ **Room Availability Management**
- 💳 **Fee Management**
- 📊 **Admin Dashboard**
- 📂 **Data Storage with MySQL**
- 🖱️ **Graphical UI with Java Swing & AWT**
